02 Templates de prompts pour le marketing

Pour le marketing digital, l'IA excelle dans : analyse de campagnes, extraction de trends, génération d'idées contenu, benchmark créatifs.

📱 Template 1 : Analyse de tendances TikTok/Instagram

Objectif : Extraire insights actionnables des trends social media

Tu es un expert social media. Analyse ces 5 trends TikTok/Instagram.

TRENDS :
[Coller descriptions de trends du TikTok Creative Center]

EXTRAIS :
1. 🎯 FORMAT DOMINANT
   - Type de vidéo (transition, storytelling, POV, dance...)
   - Durée moyenne
   - Style de montage

2. 🎵 AUDIO & MUSIQUE
   - Sons/musiques récurrents
   - Pourquoi ils fonctionnent ?

3. 💡 HOOK (accroche)
   - Quels premiers mots/images captent ?
   - Stratégies d'attention

4. 📊 ÉLÉMENTS VIRAUX
   - Points communs entre les posts performants
   - Ce qui fait réagir (likes, partages, commentaires)

5. 🎨 ADAPTATION POUR [VOTRE MARQUE]
   - 3 idées de posts inspirés de ces trends
   - Adaptations à notre ton/produit

TON : Actionnable, concret, orienté création contenu

💡 USAGE :

Utiliser après 10 min de scroll TikTok Creative Center. Copier les 5 top trends de la semaine + coller dans ChatGPT/Claude. Résultat : brief créatif prêt à l'emploi pour votre content creator.

🕵️ Template 2 : Competitive Intelligence (campagne concurrent)

Objectif : Décrypter la stratégie marketing d'un concurrent

Analyse cette campagne publicitaire de [CONCURRENT].

CONTEXTE : [Secteur, positionnement concurrent]

CRÉATIFS :
[Coller screenshots ou descriptions des ads de Meta Ad Library]

ANALYSE :

1. 🎯 STRATÉGIE IDENTIFIÉE
   - Quel message principal ?
   - Quelle promesse/bénéfice mis en avant ?
   - Quel ton (premium, accessible, fun, sérieux) ?

2. 👥 CIBLE VISÉE
   - Qui est la cible (âge, genre, intérêts probables) ?
   - Quels pain points adressés ?

3. 🎨 CRÉATIFS & FORMATS
   - Types de visuels (photo produit, lifestyle, UGC, motion...)
   - Structures des messages (AIDA, PAS, storytelling...)
   - CTA utilisés

4. 💰 INSIGHTS BUDGET/PORTÉE
   - Estimation intensité de la campagne (nombre de variantes)
   - Durée apparente
   - Zones géographiques ciblées

5. 💡 CE QU'ON PEUT APPRENDRE
   - Leurs forces à copier
   - Leurs faiblesses = nos opportunités
   - 3 actions pour nous différencier

FORMAT : Rapport structuré, factuel, orienté action
💡 Template 3 : Générateur d'idées de contenu

Objectif : Créer un planning éditorial basé sur la veille

Génère 15 idées de posts pour [PLATEFORME] basées sur ces insights.

CONTEXTE MARQUE :
- Secteur : [ex: cosmétique bio]
- Cible : [ex: femmes 25-40 ans, éco-conscientes]
- Ton : [ex: éducatif, bienveillant, accessible]
- Objectif : [ex: engagement + notoriété]

INSIGHTS DE VEILLE :
[Coller 3-5 trends/actus récentes de votre veille]

GÉNÈRE 15 IDÉES avec cette structure :

📱 FORMAT : [Carrousel / Reel / Story / Post statique]
📝 CONCEPT : [Résumé en 1 phrase]
🎯 HOOK : [Première phrase qui capte l'attention]
💬 ANGLE : [Éducatif / Divertissant / Inspirant / Promo]
🏷️ HASHTAGS : [3-5 pertinents]

CONTRAINTES :
- Mix de formats (pas que des Reels)
- 50% éducatif, 30% divertissant, 20% promo
- Au moins 3 posts qui surfent sur trends actuels

TON : Créatif, varié, aligné avec l'ADN de marque

💡 VARIANTE EXPRESS :

"Donne-moi 5 idées de Reels Instagram pour [marque] basées sur les trends TikTok de cette semaine. Format : Concept + Hook + Hashtags."

📊 Template 4 : Benchmark créatifs

Objectif : Comparer et noter des créatifs publicitaires

Compare ces 5 créatifs publicitaires et crée un benchmark.

CRÉATIFS :
[Coller 5 screenshots avec descriptions]

CRITÈRES D'ÉVALUATION (note /10 pour chaque) :

1. 👁️ ATTENTION (impact visuel immédiat)
2. 💬 CLARTÉ DU MESSAGE (compréhension en <3 sec)
3. 🎨 ORIGINALITÉ CRÉATIVE (se démarque de la masse)
4. 🎯 ALIGNEMENT CIBLE (parle à la bonne audience)
5. 🔥 DÉSIRABILITÉ (donne envie d'agir)

TABLEAU DE NOTATION :
| Créatif | Attention | Clarté | Originalité | Cible | Désirabilité | TOTAL |
|---------|-----------|--------|-------------|-------|--------------|-------|
| Créa 1  |    /10    |  /10   |    /10      |  /10  |     /10      |  /50  |
| ...     |           |        |             |       |              |       |

ENSUITE :
- TOP 3 créatifs et pourquoi
- Patterns gagnants identifiés
- 3 recommandations pour nos prochains créatifs

FORMAT : Tableau + analyse synthétique
✍️ Template 5 : Optimisation de copy publicitaire

Objectif : Améliorer un texte publicitaire existant

Optimise ce copy publicitaire pour augmenter le CTR.

COPY ACTUEL :
[Coller votre texte]

CONTEXTE :
- Produit : [Description]
- Cible : [Profil]
- Objectif : [Conversions / Clics / Notoriété]
- Canal : [Facebook Ads / Google Ads / Instagram...]

GÉNÈRE 5 VARIANTES avec :

1. VERSION ÉMOTIONNELLE
   - Joue sur les émotions (peur, joie, appartenance)
   - Storytelling court

2. VERSION BÉNÉFICES
   - Liste 3 bénéfices concrets
   - Résultats mesurables

3. VERSION URGENCE
   - Scarcité ou temps limité
   - FOMO (Fear of Missing Out)

4. VERSION QUESTION
   - Démarre par une question engageante
   - Interpelle directement

5. VERSION SOCIAL PROOF
   - Chiffres, témoignages, stats
   - Preuve de popularité

Pour chaque variante :
- Titre (max 40 caractères)
- Corps du texte (max 125 caractères)
- CTA (Call-to-Action)

+ RECOMMANDATION : Quelle variante A/B tester en priorité et pourquoi

💡 Conseils de prompting marketing

  • • Contexte marque : Toujours préciser secteur, cible, ton, objectif
  • • Exemples concrets : Montrer 2-3 exemples de ce que vous voulez
  • • Formats spécifiques : Demander tableaux, bullets, titres courts...
  • • Contraintes claires : Limites de caractères, nombre de variantes...
  • • Itération : "Rends la version 2 plus punchy" → affiner les outputs

03 Workflows Marketing : Veille + IA

3 workflows clés pour automatiser votre veille marketing avec l'IA.

🔄 Workflow 1 : Veille Social Media Hebdo (1h/semaine)

1. Collecte (20 min)

• TikTok Creative Center : top 10 trends

• Instagram Explore : formats viraux

• Screenshot + notes rapides

2. Analyse IA (20 min)

• ChatGPT : Template "Analyse trends"

• Extraire formats + hooks + audio

• Générer adaptations marque

3. Idéation (15 min)

• Template "Générateur idées"

• 10-15 concepts de posts

• Brief créatifs ready

4. Planning (5 min)

• Ajouter à Notion/Trello

• Assigner content creator

• Dates de publication

📋 OUTPUT : Trend Report + 15 idées de contenu prêtes

Au lieu de passer 3h à analyser manuellement, l'IA vous donne les insights en 20 min. Vous économisez 2h+ par semaine = 8h/mois = 1 journée de travail !

🕵️ Workflow 2 : Competitive Intelligence Mensuel (2h/mois)

Phase 1 : Collecte (30 min)

  • • Meta Ad Library : 10-15 ads concurrents
  • • Google Alerts : actualités du mois
  • • Social media : nouveaux contenus
  • • Site web : nouveaux produits/offres

Phase 2 : Analyse (60 min)

  • • Template "Competitive Intelligence"
  • • Analyse de chaque campagne
  • • Benchmark créatifs
  • • Extraction patterns gagnants

Phase 3 : Rapport (30 min)

  • • Synthèse insights clés
  • • Opportunités identifiées
  • • Recommandations actions
  • • Présentation équipe
⚡ Workflow 3 : Optimisation Copy (30 min/campagne)

Pour chaque nouvelle campagne pub, générer et A/B tester plusieurs variantes de copy

Étape 1 : Version initiale (5 min)

Rédiger copy v1 manuellement

Étape 2 : IA génère variantes (10 min)

Template "Optimisation copy" → 5 variantes

Étape 3 : Sélection (10 min)

Garder top 3 pour A/B tests

Étape 4 : Lancement (5 min)

Setup tests dans Ads Manager

Résultat : Au lieu de passer 2h à brainstorm manuellement, vous avez 5 variantes testables en 10 min. Les tests révèlent souvent +20-50% de CTR sur la meilleure variante.
